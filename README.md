# 🚀 Smart Job Portal

A Full Stack Job Portal Application built using React.js, Spring Boot, JWT Authentication and MySQL.

---

## 📌 Features

### 👨 User Features

- User Registration
- User Login
- JWT Authentication
- Search Jobs
- Filter Jobs by Category
- Filter Jobs by Location
- Sort Jobs by Salary
- Apply for Jobs
- Save Jobs
- View Job Details
- Dark Mode
- Profile Page

---

### 👨‍💼 Admin Features

- Admin Login
- Add Job
- Edit Job
- Delete Job
- Upload Company Logo
- Dashboard
- Pagination

---

## 🛠 Tech Stack

### Frontend

- React.js

### Backend

- Spring Boot
